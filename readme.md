Інструкція:
1) npm install - підтягнути всі залежності
2) npm run serve - to get a ready local server 
3) npm run build - не треба робити, бо я скомпільовані файли запушив
4) npm run test - запустити тести

1) Використовував bootstrap 4, бо bootstrap 5 погано підтримується by IE-10, є https://coliff.github.io/bootstrap-ie11/ , але вона не вирішує всіх проблем + це тільки для 11 версії
2) Перевіряв через в Microsoft Edge (через мод IE-10), на чистому Internet Explorer не міг перевірити, бо не знайшов олдскульного віндовса :)
3) Покрив тестами тільки те, що має сенс, хоча можна було б більше
4) Для компілювання стилів і транспайлінгу в ES5, використовував Webpack + Babel + postcss
5) Верстку не робив pixel perfect, бо не було такої вимоги, хоча старався максимально наближено зверстати, можливо, поінде буде різниця в піксель
